Introduction

JavaScript is a cross-platform, object-oriented scripting language. It is a small and lightweight language. Inside a host environment (e.g., a web browser), JavaScript can be connected to the objects of its environment to provide programmatic control over them.

JavaScript contains a standard library of objects, such as Array, Date, and Math, and a core set of language elements such as operators, control structures, and statements. Core JavaScript can be extended for a variety of purposes by supplementing it with additional objects:

  • Client-side JavaScript extends the core language by supplying objects to control a browser and its Document Object Model (DOM). For example, client-side extensions allow an application to place elements on an HTML form and respond to user events such as mouse clicks, form input, and page navigation.
  • Server-side JavaScript extends the core language by supplying objects relevant to running JavaScript on a server. For example, server-side extensions allow an application to communicate with a database, provide continuity of information from one invocation to another of the application, or perform file manipulations on a server.

This guide assumes you have the following basic background:

  • Good working knowledge of HyperText Markup Language (HTML).
  • Some programming experience. If you are new to programming, try one of the tutorials linked on the main page about JavaScript.
JavaScript and Java

Java and JavaScript are similar in some ways but fundamentally different in others. The JavaScript language resembles Java but does not have Java's static typing and strong type-checking. JavaScript follows a syntax similar to Java's and is widely used in web applications. Unlike Java's language features, JavaScript supports a range of programming paradigms, including object-oriented and functional programming.

JavaScript engines interpret and execute JavaScript code. Modern browsers contain built-in JavaScript engines that execute JavaScript directly on the browser.

Hello world

To get started with JavaScript, you can try the following example to print “Hello, World!” to the console:

console.log('Hello, World!');

This will print “Hello, World!” to the console in most web browsers.

Variables

We can declare variables in three ways:

var x = 5;

Using let or const is generally preferred in modern JavaScript due to better scoping rules.

Data types

The latest ECMAScript standard defines seven data types:

  • Six data types that are primitives:
    • Boolean
    • Null
    • Undefined
    • Number
    • String
    • Symbol
  • And one type: Object
if...else statement

The if statement executes a statement if a specified condition is true. If the condition is false, another statement can be executed using the else statement.

if (condition) { // statement } else { // statement }
while statement

The while statement creates a loop that is executed while a specified condition evaluates to true.

while (condition) { // statement }
Function declarations

A JavaScript function is defined with the function keyword, followed by a name, followed by parentheses ().

function myFunction() { return "Hello, World!"; }
